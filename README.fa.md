### مهندس نرم‌افزار • توسعه‌دهنده دات‌نت • بنیان‌گذار فرادا

من نرم‌افزارهای مدرن را برای **وب، دسکتاپ، موبایل و بک‌اند** با استفاده از اکوسیستم Microsoft .NET طراحی و توسعه می‌دهم.

تمرکز من بر توسعه نرم‌افزارهای مقیاس‌پذیر، معماری تمیز، کدنویسی استاندارد و خلق تجربه‌های کاربری باکیفیت است.
